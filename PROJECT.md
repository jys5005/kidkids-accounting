# 어린이집 재무회계 (수전자장부)

포트 4000에서 독립 운영되는 어린이집 재무회계 시스템

## 기술 스택
- Next.js 16 (App Router)
- Tailwind CSS 4
- Puppeteer (키즈키즈 스크래핑)
- xlsx (엑셀 내보내기)
