# 수전자장부 - 어린이집 회계관리시스템

## 프로젝트 구조
- **프레임워크**: Next.js 16, App Router (`src/app/`)
- **스타일**: Tailwind CSS
- **언어**: TypeScript
- **포트**: 3000

## 주요 경로
- `/accounting` - 회계현황 (메인)
- `/budget/*` - 예산관리 (작성, 과목전용, 보고서, 집행현황, 보육통합예산보고)
- `/voucher/*` - 전표관리 (입력, 삭제전표, 잔액비교, 거래내역, 계좌내역, 영수증(CIS)연동)
- `/monthly-report/*` - 월회계보고 (전표검증, 재무회계분석자료)
- `/settlement/*` - 결산관리 (월별결산서, 연말결산서, CIS결산보고)
- `/reconciliation/*` - 정산서관리 (정부보조금명세서, 보조금정산서, 누리과정정산서, 급식비정산서, 특별활동비보고서, 기타필요경비보고서, 필요경비정산서)
- `/staff/*` - 교직원 (정보, 출근부, 증명서, 공문서, 급여대장, 4대보험, 연말정산)
- `/children/*` - 아동 (아동관리, 보육료납부, 출석부)
- `/supplies/*` - 물품 (물품관리)
- `/community/*` - 커뮤니티 (게시판, 자료실, 계산기)
- `/settings/*` - 설정 (결제방식, 적요코드매칭, 거래처, 4대보험요율, 보육통합인증키)
- `/data-migration` - 데이터이관

## 개발
```bash
npx next dev -p 3000
```

- **1단**: 로고(어린이집회계관리시스템/수전자장부) + 카테고리탭(회계/교직원/아동/물품/커뮤니티) + 통합e + 어린이집명(회원정보팝업) + 사용자명 + 30분타이머 + 로그아웃
- **2단 GNB**: 노란 바(`#f5b800`), 카테고리별 메뉴 박스 + 펼쳐보기/접기
- **3단**: 하위 서브탭 (hover 시 rotateY 15도 CSS transition)

## 전표입력 (`/voucher/input`)
- **페이지**: `src/app/voucher/input/page.tsx`

### 상단 필터
- 출납년월, 일자(캘린더), 계정과목(전체▼/수입▼/지출▼), 입력구분, 금액(최소~최대), 검색(적요/계정/결제방식/전표번호), 조회, 초기화
- 라벨은 모두 왼쪽 정렬 (셀렉트/인풋과 같은 줄)

### 요약 데이터
- 컬럼설정 위에 블록 방식: 수입(합계/전표/정상/반납/삭제), 지출(합계/전표/정상/반납/삭제), 회계잔액, 계좌잔액+일치뱃지

### 기능키 툴바
- **컬럼설정** | **기능키** (토글)
- **전표**: 등록, 합산, 삭제, 일괄분리, 미계정전환
- **적요**: 삭제, 치환, 세목추가
- **매핑**(핑크): 원아경비, 거래처.적요.결제방식
- **정렬**(초록): 수입부우선, 전표번호, 전체 — 클릭 시 실제 정렬 + 상태 표시
- **출력**: 엑셀, 인쇄, 저장

### 입력방법 탭
- **간편등록**: 수기/엑셀 모드, 날짜·적요·금액·계정과목 빠른 입력, 저장 시 일괄수정으로 전송 후 초기화
  - 수기: 테이블 형태, 마이크 음성입력, 커스텀 계정과목 드롭다운(수입=파랑 호버, 지출=빨강 호버)
  - 엑셀: 드롭다운(보육통합엑셀/기본샘플/CMS/은행별) + 파일선택 + 등록, 참고사항/입력안내 말풍선
  - 간편등록 헤더+테이블 헤더 sticky (layout.tsx overflow-auto 제거)
- **건별등록**: 상단 1건 편집 박스 + 하단 전표목록, 행 클릭으로 선택(1건만), 복사/분리/반납/수수료 컬럼 숨김
- **상세등록**: 전체 항목 폼 입력 (결제방식/증빙 포함)
- **일괄수정**: 기존 스프레드시트 테이블, 클릭 편집

### 컬럼설정
- `columnOrder` state 기반 동적 렌더링 (순서변경 ▲▼ + 표시/숨기기)
- 그룹핑: 증빙(영수/이체/은행/증빙), 금액(수입/지출/잔액), 계정(복사/계정과목/세목/계정코드/분리/반납)
- 고정필드: 번호/일자/구분/적요/금액그룹/계정그룹 (체크박스 없이 `-` 표시)
- 수수료: 독립 컬럼 (건별등록에서는 숨김)
- 컬럼설정 변경 시 실시간 반영 (적용 버튼 없음)

### 테이블 헤더
- 골드 테마: `bg-[#fffbeb]` + 얇은 노란 하단선
- sticky 헤더 (스크롤 시 고정)
- `table-layout: fixed`, `minWidth: 1400px`
- 건별등록: 박스 테두리 스타일 (`single-input-mode` CSS)

### 구분 컬럼
- 입력구분 값 동기화 (은행/수기/일괄/엑셀/분리/합산)
- 색상: 은행=검정, 수기=파랑, 일괄=주황, 엑셀=노랑, 분리=보라, 합산=녹색
- 테두리+글자 같은색, 배경 흰색, 둥근 박스

### 반납 기능
- 반납 컬럼: 전표/계정 버튼
- 전표 클릭: 금액 반대 컬럼 이동(마이너스) + 계정 초기화 + 행 배경 연한 빨간색
- 다시 클릭(반납 상태): 원복 (양수로, 원래 컬럼)
- 계정 클릭: 계정과목/세목/계정코드만 초기화

### 잔액 컬럼
- 수입/지출 다음에 위치, 누적 계산 (수입+, 지출-)
- 검은색 텍스트, 수정 불가

### 증빙 아이콘
- **영수**: 카메라 아이콘 (사진 촬영)
- **이체**: 이체증 서류 아이콘
- **은행**: 클립보드 서류 아이콘
- **증빙 뱃지**: 파비콘 이미지 사용
  - 홈택스(세금계산서/계산서/현금영수증): `/hometax-favicon.ico` + 배경색 구분(파랑/노랑/초록)
  - 쿠팡/네이버/11번가/지마켓/옥션/오아시스: 각 쇼핑몰 파비콘
  - 4대보험: `/nhis-favicon.ico` (국민건강보험)

### 계정복사 기능
- **복사** 클릭 → 해당 행 파란 활성화
- 같은 구분 다른 행 → **붙임** 버튼 (클릭 시 계정과목 복사 적용, 같은 계정이면 숨김)
- 반대 구분 + 금액 있는 행 → 버튼 숨김
- 계정과목 선택 시 자동으로 복사 활성화
- 외부 클릭 시 copySelected 초기화

### 계정코드
- 계정과목 → 코드 자동 매핑 (`accountCodeMap`, `subAccountCodeMap`)
- 코드 → 계정과목 역매핑 (`codeToAccount`)
- 클릭 시 빈 입력 (기존 코드 placeholder로 회색 표시)
- 엔터/탭 → 아래 행 이동, 매칭 코드면 계정과목 자동 업데이트, 안 맞으면 원래 코드 복원
- 수입=파랑, 지출=빨강 글자색 (계정과목과 동일)

### 계정과목 선택 팝업
- 셀 클릭 시 우측 팝업 (상위 3행은 아래로, 나머지는 위로)
- 목: 라벨만 표시, 세목: [세목] 뱃지 박스 표시
- 세목 선택 시 부모 목 자동 탐색 → account/subAccount 동시 업데이트

### 행 스타일
- 기본 흰색, 호버 시 수입=`bg-blue-100`, 지출=`bg-red-100`
- 셀 편집(클릭) 시 배경색 유지
- 날짜 변경 시 굵은 골드 구분선, 같은 날짜는 얇은 골드 선

### 입력구분 옵션
- 은행, 수기, 일괄, 엑셀, 분리, 합산

### 결제방식
- "지출-"/"수입-" 접두사 표시에서 자동 제거
- 가운데 정렬

### 담당자 정보
- Header 3단(하위탭) 오른쪽에 표시
- 회계담당/급여관리/팩스 (샘플 데이터)

### 하단 합계
- 블록 방식: 수입(파란박스), 지출(빨간박스), 잔액(초록박스)

## 공통 계정과목 데이터 (`src/lib/accounts.ts`)
- 모든 페이지에서 import하여 사용 (페이지별 하드코딩 금지)
- `incomeAccounts`, `expenseAccounts` — 목+세목 배열 (isSub: true)
- `accountCodeMap` — 계정과목명 → 4자리 코드
- `subAccountCodeMap` — 세목명 → 5자리 코드
- `codeToAccount` — 코드 → {account, subAccount} 역매핑
- `isIncomeAccount()` — 수입 여부 판별
- **계정과목 표시 규칙**: 항상 목 박스 + 세목 박스 별도 컬럼 분리
- **색상**: 수입=파란(text-blue-700), 지출=빨간(text-red-600)
- **드롭다운**: 커스텀 팝업, 목/세목 뱃지(배경색+흰글씨), 외부 클릭 닫기

## 건별등록/상세등록 공통 프로세스
- 하단 테이블: 읽기 전용, 클릭/체크로 전표 선택만
- 상단 패널: draftRow 기반 편집 (원본 직접 수정 안 함)
- 수정 버튼: draftRow → rows 반영, 삭제 버튼: 행 제거
- draftRow 동기화: inputMode/checked 변경 시만 (editingCell 변경 시 덮어쓰기 방지)
- 상세등록 드롭다운: `detailDropdown` state 사용 (editingCell과 분리)

## 잔액비교 (`/voucher/balance`)
- **페이지**: `src/app/voucher/balance/page.tsx`
- **회계잔액 비교** 타이틀 + **참고** 말풍선(우측, 호버)
- **서브탭**: 월별 통장잔고 비교(+조회년도) / 일별 통장잔고 비교(+조회월)
- **테이블**: 입금일자 / 계좌잔액(은행별) / 계좌합계잔액(A)(계좌거래내역합계) / 회계잔액(B)(회계내역합계) / 비교차액(A)-(B)(거래내역차액) / 확인(정상/불일치+말풍선)
- **일별**: 전표/통장 비교 컬럼(보기 버튼), 금액 밑에 당일 거래내역 합계 (괄호)
- **보기 팝업**: 현금출납부 vs 통장거래 좌우 비교, 전체/일치/불일치 필터(상단), 불일치 행 빨간 배경
- **거래 매칭**: 금액 일치 + 적요 2자 이상 연속 일치 (`matchSummary`, `findMatch`)
- 회계연도: 3월~다음해 2월 (12행)

## 거래내역 (`/voucher/transactions`)
- **페이지**: `src/app/voucher/transactions/page.tsx`
- **뷰 전환 버튼**: 거래조회 / 현금출납부 / 총계정원장 / 계정과목별총괄표 / 월별수입지출합계 / 합계잔액시산표 / 월별비교
- **거래조회**: 출납기간(날짜범위) + 이번달/이전달/과년도 버튼 + 계정과목(커스텀드롭다운) + 결제방식 + 금액(숫자만+콤마) + 품목(적요) + 조회
- **현금출납부**: 출납연월(월 드롭다운) + ☑구간별(체크 시 년월~년월 + 3개월/6개월/과년도 버튼)
- **총계정원장**: 계정과목별 그룹핑(목+세목), 전월이월, 세목별 소계/목 소계/총합계, 수정 버튼→수정 팝업(발의일자/발행일자/계정과목+세목/수입계정/지출계정/금액/적요/거래처/결제방식), 원장전체보기 체크
- **계정과목별총괄표**: 년도 드롭다운 + ☑구간별, 기본=월별 피벗(계정코드/계정과목(목+세목)/예산액/총합계/비율/3월~2월/합계, 수입/지출 섹션 분리), 구간별=날짜범위 입금/출금/잔액 요약(max-w-4xl)
- **월별수입지출합계**: 년도 드롭다운, 월구분/수입액/지출액/차이액/월잔액(누적)+누계행 (max-w-4xl)
- **합계잔액시산표**: 출납년월 드롭다운(구간별 없음), 수입누계/수입합계/계정과목(가운데)/지출합계/지출누계 (max-w-4xl)
- **월별비교**: 출납년월 드롭다운(구간별 없음), 계정과목별 2달 좌우 비교(기준월/다음월), 목 뱃지 헤더
- **요약**: 수입합계/지출합계/전표개수(수입N/지출N)
- **출력**: 뷰별 버튼 분기 — 거래조회(결의서/전체내역+엑셀/선택내역+엑셀/푸르니), 현금출납부(인쇄/엑셀), 총계정원장(거래처+엑셀/증빙서번호+엑셀/거래처[총괄]/증빙서번호[총괄])

## 정산서관리
### 정부보조금명세서 (`/reconciliation/subsidy`)
- 회계연도/발행기간 검색, 보조금합계 + 말풍선(기관보육료,연장보육료,그밖의지원금,자본보조금)

### 보조금정산서 (`/reconciliation/settlement`)
- 보조금거래 ↔ 일반거래 이동 (체크 선택 → 이동)
- 제목 입력 + 저장, 조회기간/수입금액/지출금액

### 누리과정정산서 (`/reconciliation/nuri`)
- 3모드: 회계연도(3~2월) / 해당연도(1~12월) / 상세정산서
- 세입 2계정 + 세출 15계정 × 12개월, 월합계/누계

### 급식비정산서 (`/reconciliation/meal`)
- 공동구매/개별구매급식비, 합계, 공동구매비율(%)
- 단축키: 지난달/과년도, 말풍선(청정급식비 안내)

### 특별활동비보고서 (`/reconciliation/activity`)
- 조회연도/구간별, 상반기/하반기, 그밖의지원금 포함 체크

### 기타필요경비보고서 (`/reconciliation/expense`)
- 세입/세출/정산 3섹션 × 7항목(기타필요경비~기타시도특성화비)
- 조회연도/구간별, 그밖의지원금 포함 체크(기타필요경비에 합산)

### 필요경비정산서 (`/reconciliation/required-expense`)
- 계정별 상세 필터 (수입/지출 계정, 인천 법정/만5세아/외국인 포함)
- 원아매칭, 전표 이동, 삭제/분리

## 예산작성 (`/budget/create`)
- **관-항-목 테이블**: 관/항 병합 표시, 목/세목 인라인 세부항목
- **산출식**: 단가*수량*개월 입력 → onBlur 시 자동 계산, %는 /100 변환
- **합산**: 세부항목 → 목 → 항 → 관 자동 합산
- **세입/세출 탭**: 코드 충돌 방지 (세출 E접두사, 표시 시 제거)
- **추경**: 추경하기 → 날짜 선택 → N차 추경 생성 (본예산 데이터 복사, 별도 state)
- **상태**: 신청/작성중/작성완료(잠금), 예산등록/삭제 버튼
- **버튼**: 전달사항, 이전예산호출, 엑셀업로드, 일괄변경(말풍선)
- **단위**: 원/천원 라디오, 인쇄/엑셀 아이콘
